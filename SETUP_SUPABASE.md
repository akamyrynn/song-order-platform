# Быстрая настройка Supabase

## Шаг 1: Создать таблицы

1. Открой https://supabase.com/dashboard/project/ejnbpcqmfyorppjkajpp
2. Слева нажми **SQL Editor**
3. Нажми **New query**
4. Скопируй ВСЁ содержимое файла `supabase-migration.sql`
5. Вставь в редактор
6. Нажми **Run** (или F5)

## Шаг 2: Проверить что работает

1. Запусти сервер:
```bash
npm run dev
```

2. Открой в браузере:
```
http://localhost:3002/api/test-db
```

Должен вернуться JSON с `success: true`

## Шаг 3: Проверить конструктор

1. Открой http://localhost:3002/constructor
2. Заполни форму
3. Нажми "Создать заказ"

Должно работать!

## Что изменилось

- Убрал Prisma полностью
- Теперь используется только Supabase клиент
- Не нужен пароль от базы данных
- Работает с теми ключами что ты уже дал (anon key и service role key)

## Если не работает

Проверь что:
1. Таблицы созданы в Supabase (шаг 1)
2. В `.env` правильные ключи (они уже там есть)
3. Сервер перезапущен после изменений
